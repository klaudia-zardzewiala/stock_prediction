# Stock prediction
This program predicts stock prices by using MLM for forecasting stock prices.

## Table of Contents
1. Introduction
2. Features
3. Technologies used
4. Installation

## 1. Introduction
This project demonstrates how to build a simple stock price prediction model using Python and machine learning.
The goal is to use historical price data to predict future closing prices and visualize the prediction results.

## 2. Features
- Downloading stock market data
- Adjusted Close Price
- Training a machine learning model (LLR, SVM)
- Testing a machine learning model (LLR, SVM)
- Making predictions for future prices for n days

3. Technologies used
- Python 3
- yfinance
- numpy
- sklearn
